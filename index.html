How to create a Telegram Bot (first time setup)
Step 1 — Create your bot:
1. Open Telegram and search @BotFather
2. Send the message: /newbot
3. Choose a name (e.g. Fuel Dispatch Bot)
4. Choose a username ending in bot (e.g. myfleet_fuel_bot)
5. BotFather gives you a token like 123456:ABCdef... — copy it below
Step 2 — Get your Chat ID:
1. Send any message to your new bot from your driver's phone
2. Open this URL in a browser (replace TOKEN):
https://api.telegram.org/botTOKEN/getUpdates
3. Copy the "id" number from chat object — that's your Chat ID
4. For a group: add the bot to the group, then get the group's negative ID
Token and Chat ID are saved in your browser — you only set this up once per device.
